## 列表推导式, 生成器表达式, 字典推导式
- 简单的for循环都可以用推导式进行完成，性能高比较高，但是如果场景过于复杂就不要用，可读性要排在第一位.

```
# 列表生成式的性能高于列表操作
list1 = [i for i in range(20) if i % 2 == 1]
list2 = [i * i for i in range(10)]
print(list1)
print(list2)

# 生成器表达式
gen1 = (i for i in range(20) if i % 2 == 1)
print(type(gen1))
list3 = list(gen1)
print(list3)

# 字典推导式
b={"boby1":30,"boby2":60}
dict={value:(key+1) for value,key in b.items() }
print(dict)
```
